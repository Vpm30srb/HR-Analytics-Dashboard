# HR Analytics Dashboard

## 📊 Project Overview

This project is an interactive HR Analytics Dashboard developed using **Power BI** to analyze employee attrition, salary distribution, employee demographics, and workforce trends.

The dashboard helps HR teams identify the major reasons behind employee attrition and supports data-driven decision making.

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV

---

## 📈 Key Performance Indicators (KPIs)

- Overall Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Years at Company

---

## 📊 Dashboard Features

- Employee Attrition Analysis
- Salary Slab Analysis
- Education Field Analysis
- Age Group Analysis
- Job Role Analysis
- Years at Company Analysis
- Department Filter

---

## 💡 Key Insights

- Employees earning below **5K** show the highest attrition.
- Employees aged **26–35** have the highest attrition.
- Laboratory Technicians and Sales Executives experience the highest attrition.
- Most employees leave within the first year.
- Life Sciences has the highest attrition among education fields.
